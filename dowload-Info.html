Configure sua conta no Arduino Cloud!

Aprenda a criar sua conta no Arduino Cloud e desenvolver seu primeiro código.
Passo a passo detalhado para iniciantes.

O que é o Arduino Cloud

O Arduino Cloud é uma plataforma que permite a programação, compilação e armazenamento de código na nuvem. Ele facilita o desenvolvimento de projetos IoT, oferecendo um ambiente colaborativo e acessível de qualquer lugar.


Passo a Passo

Siga os passos abaixo para criar sua conta no Arduino Cloud e desenvolver seu primeiro código.

1. Criar Conta

Acesse o site do Arduino Cloud: Arduino Cloud. Clique em "Sign Up" e preencha os dados solicitados. Confirme seu e-mail para ativar a conta.

2. Configurar Dispositivo

Após criar a conta, faça login e acesse a seção "Devices". Clique em "Add Device" e siga as instruções para conectar seu Arduino. Escolha o modelo do seu Arduino e siga os passos para configurá-lo.

3. Criar Primeiro Código

Acesse a seção "Sketches" no Arduino Cloud. Clique em "New Sketch" e escreva o seguinte código para piscar um LED: 

void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);
  delay(1000);
  digitalWrite(LED_BUILTIN, LOW);
  delay(1000);
}
Salve o sketch e clique em "Upload" para enviar o código ao seu Arduino.
