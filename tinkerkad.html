Configure sua conta no Tinkercad!

Passo a passo detalhado para iniciantes.

O que é o Tinkercad

O Tinkercad é uma plataforma online gratuita que permite criar e simular circuitos eletrônicos, incluindo projetos com Arduino. Ele é ideal para iniciantes e oferece uma interface intuitiva para facilitar o aprendizado.


Passo a Passo

Siga os passos abaixo para criar sua conta no Tinkercad e desenvolver seu primeiro projeto em Arduino.

1. Criar Conta

Acesse o site do Tinkercad: Tinkercad. Clique em "Join Now" ou "fazer login" e preencha os dados solicitados. Confirme seu e-mail para ativar a conta.



2. Navegar pelo Dashboard

Após criar a conta, faça login para acessar o dashboard do Tinkercad. No dashboard, você verá várias opções:

  • Gallery: Explore projetos criados por outros usuários.
  • Learn: Acesse tutoriais e recursos de aprendizagem.
  • Circuits: Crie e simule circuitos eletrônicos.
  • 3D Designs: Desenvolva e visualize projetos 3D.



3. Criar Novo Projeto

No dashboard, clique em "Circuits" e depois em "Create new Circuit" para começar um novo projeto. Arraste e solte os componentes necessários, como o Arduino e um LED, para montar seu circuito.

4. Escrever Código Arduino

No ambiente do Tinkercad, clique no Arduino e selecione "Code" para abrir o editor de código. Escreva o seguinte código para piscar um LED: 

void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);
  delay(1000);
  digitalWrite(LED_BUILTIN, LOW);
  delay(1000);
}
Clique em "Start Simulation" para testar seu código e ver o LED piscando.
