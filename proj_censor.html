Meça sua temperatura!

aprenda a fazer seu proprio termômetro!
Sem enrolação.







Criar um termômetro com Arduino é um projeto relativamente simples e popular que permite medir a temperatura ambiente usando um sensor de temperatura, como o LM35, DHT11 ou DS18B20, e exibir o valor em um display. Aqui está um guia passo a passo usando o sensor LM35 e um display LCD 16x2:


Componentes Necesarios

  • 1x buzzer
  • 9x jumpers machos
  • 1x arduino
  • 1x led
  • 1x resistor
  • 1x protobord
  • 1x Sensor de Temperatura LM35

Determinando o Resistor Adequado para LEDs



Para determinar o valor do resistor para LEDs, você pode usar a tabela prática de resistores. No entanto, é mais preciso verificar o datasheet do fabricante do LED para obter os valores exatos de tensão e corrente. Veja mais detalhes em Como calcular o resistor adequado para o LED. Resumidamente, resistores acima de 150 Ω são geralmente seguros para LEDs de todas as cores em circuitos de até 5V.


ATENÇÃO







Lembre-se que o LED tem polaridade: O terminal maior tem polaridade positiva e o lado do chanfro tem polaridade negativa

Passo 1

faça a conexão da porta digital do Arduino no terminal positivo (anodo) e o GND no terminal negativo (catodo) do led.
Para evitar danos ao led é necessário a inclusão de um resistor no circuito. Como o resistor é um limitador da corrente elétrica, ele poderá estar conectado no anodo (terminal maior) ou no catodo (terminal menor) do led.

Passo 2

Valores Comuns de Resistores

  • LEDs Vermelho, Laranja e Amarelo: 150 Ω
  • LEDs Verde e Azul: 100 Ω ou 150 Ω

6° O buzzer tem polaridade: se retirar o adesivo superior do buzzer poderá ver um sinal de positivo (+). Este sinal mostra onde está o pino positivo do componente.

7° A montagem abaixo foi realizada em um protoboard com linhas de alimentação contínuas. Verifique se o seu protoboard possui linhas de alimentação contínuas ou separadas - saiba mais em protoboard

Visão geral

Programa

/*******************************************************************************
*
* Projeto 11 – Sensor de temperatura com alarme
*
*******************************************************************************/

const int LM35 = 0; // entrada sensor no Arduino: A0 - Analógica
float temperatura = 0; // variável tipo float - inícia no 0
int ADClido = 0;
const int Buzzer = 12; // entrada buzzer: 12 - digital
const int Led10 = 10; // entrada led: 10 - digital

//****************************************************************************

void setup(){
Serial.begin(9600); // taxa comunicação da placa com o computador
analogReference(INTERNAL); //No Mega, use INTERNAL1V1, no Leonardo remova essa linha
pinMode(Buzzer, OUTPUT);
pinMode(Led10, OUTPUT);
}

//****************************************************************************

void loop(){
ADClido = analogRead(LM35);
temperatura = ADClido * 0.1075268817204301; //no Leonardo use 0.4887585532
Serial.print("Temperatura = "); //mostra valor na tela
Serial.print(temperatura);
Serial.println(" *C");

if(temperatura > 32){ // setado como 32ºC
digitalWrite(Buzzer, HIGH); //aciona o buzer
digitalWrite(Led10, HIGH); //aciona o led
}
else{
digitalWrite(Buzzer, LOW);
digitalWrite(Led10, LOW);
}
delay(500);
}
